# Summary of Expenses

## Overview

The reMIND project was completed without incurring any direct financial expenses. All tools, technologies, and resources used throughout the project were either free, open-source, or provided through university-supported programs.

---

## Expense Breakdown

| Category           | Description                                      | Cost |
|--------------------|--------------------------------------------------|------|
| Hardware           | No additional hardware purchased                 | $0   |
| Software           | Open-source tools and university-provided access | $0   |
| Cloud Services     | Microsoft Azure (student credits)                | $0   |
| Development Tools  | Xcode, VS Code (free tools)                      | $0   |
| Design Tools       | Free tools (e.g., Figma/Canva if used)           | $0   |
| Miscellaneous      | No additional materials or services required     | $0   |

---

## Donated / Provided Resources

The following resources were utilized at no cost:

- Microsoft Azure Student Credits  
  Used for AI services, cloud infrastructure, and database hosting.

- Apple Development Tools (Xcode, watchOS SDK)  
  Provided free for development and testing of the Apple Watch application.

- Open-Source Libraries and Frameworks  
  Used for backend development, APIs, and system integration.

---

## Justification

The project was intentionally designed to be cost-efficient and accessible by leveraging free and widely available tools. This aligns with the goal of creating a scalable and affordable solution for real-world deployment, particularly in healthcare contexts where cost can be a barrier to adoption.

---

## Summary

- Total Project Expenses: $0  
- Total Donated Resources: Significant (cloud credits, software tools)

The absence of direct expenses demonstrates the feasibility of developing a functional, scalable AI healthcare solution using accessible technologies.
