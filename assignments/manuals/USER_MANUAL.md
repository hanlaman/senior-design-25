# Full User Manual

This manual provides a complete reference for both patients and caregivers.

---

## User Roles

| Role | Responsibilities |
|-----|------------------|
| Patient | Wear and interact with device |
| Caregiver | Manage settings, reminders, alerts |

---

## Privacy & Data Use

- Patient data is encrypted
- Only approved caregivers can access information
- Emergency data is shared only when needed

---

## Battery & Maintenance

- Battery life: ~24 hours
- Charge daily
- Clean device with dry cloth

---

## Error Messages

| Message | Meaning |
|-------|--------|
| “Disconnected” | Bluetooth lost |
| “Low Battery” | Recharge device |
| “No Internet” | App features limited |
